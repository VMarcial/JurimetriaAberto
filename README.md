# Jurimetria

Este programa faz 3 funções

- Baixa metadados e os processos em PDF da segunda instância do TJSP, selecionando em especial a parte resumo do processo.
- Passa esses programas por uma biblioteca de NPL.
- Utiliza o output do NPL em conjunto com os metadados como insumo para uma rede neural que busca prever o resultado do processo.
